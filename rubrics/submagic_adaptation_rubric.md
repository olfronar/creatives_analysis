# Submagic Adaptation Rubric

Use this rubric to judge whether an adaptation idea is useful, safe, and executable.

| Dimension | What strong looks like |
|---|---|
| Strategy transformation | Preserves the competitor mechanism but changes expression, proof, script, visuals, and brand context. |
| Submagic feature mapping | Each variant names features such as AI Captions, Magic Clips, Auto Edit, B-roll, Magic Zoom, Brand Kit, templates, publishing, or API. |
| Creator/agency relevance | The pain maps to editing time, output volume, brand consistency, client work, or platform publishing. |
| Script usefulness | Includes hook, promise, beats, captions, visual notes, and CTA. |
| Platform adaptation | Notes how TikTok, Reels, Shorts, Meta, or LinkedIn versions should differ. |
| Evidence lineage | Explains which competitor mechanism inspired the idea without copying the asset. |
| Copycat safety | Avoids mimicry of protected expression, branded style, exact wording, music, creator likeness, and unverifiable claims. |
| Testability | Includes a clear creative hypothesis such as hook, caption style, proof sequence, or CTA timing. |
