# Short-Form Creative Rubric

Scores use a `0-3` scale.

| Score | Meaning |
|---:|---|
| 0 | Missing, unclear, or actively harmful. |
| 1 | Present but weak, delayed, generic, or hard to understand. |
| 2 | Functional and clear, with some room for sharper execution. |
| 3 | Strong, specific, memorable, and supported by visible evidence. |

## Dimensions

| Dimension | Score evidence required |
|---|---|
| Hook strength | First-frame read, opening line, visible problem, surprise, proof, or pattern interrupt. |
| Promise clarity | Timestamp where the payoff becomes understandable. |
| Emotional arc | Evidence of movement from pain/desire to relief/control/status. |
| Brand/product integration | First product/brand appearance and whether it supports the story. |
| Visual memorability | Distinct visual motif, face, motion, UI moment, transformation, or contrast. |
| Message hierarchy | Alignment between captions, voiceover, on-screen text, and visuals. |
| Pacing/edit rhythm | Cut cadence, dead seconds, repeated shots, comprehension load. |
| Core story/idea | The central transformation, reveal, demo, contradiction, or proof mechanism. |
| CTA strength | Specificity, timing, and connection to the promise. |
| Likely drop-off risk | Timestamped moments where attention or comprehension may break. |
| Proof credibility | Demo, result, comparison, social proof, or believable mechanism. |
| Caption readability | Size, contrast, safe zones, sync, and mobile legibility. |
| Audience fit | Fit for creators/agencies and their editing/output-volume pain. |
| Copycat/IP risk | Whether adaptation risks copying protected expression instead of mechanism. |
